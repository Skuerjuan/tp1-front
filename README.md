# TP1 Front-end
## Felipe Maierowicz 5°D
## Conversor de Unidades
### Construido en Astro, integrado con Tailwind
---
## Puntos de la consigna:
- Cuenta con 3 páginas distintas, con la funcionalidad de convertir unidades usando JavaScript
- Los componentes Converter, Navbar y SelectorUnidades fueron codeados con modularidad en mente
- Los botones cambian de color con el hover
- Usa Tailwind
- Usa un fetch a una API en la pantalla principal 
---
### La página web se puede ejecutar al descargar las dependencias necesarias con 'npm i' y después usando 'npm run dev'.
